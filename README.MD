# calendarioInteligente en telegram
bot para organizar tu calendario en telegram

## Pasos para ejecutar
### Paso 1:
- Crear un virtual environment con el comando `python -m venv venv`

### Paso 2:
- Instalar los requerimientos con el comando `pip install -r requirements.txt`

### Paso 3:
- Crear un archivo `.env` y colocar la apiKey (contactar con administradores)
